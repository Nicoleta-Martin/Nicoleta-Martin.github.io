This is a smart device designed to monitor the air quality in my room. It is very important to see how TVOC values are changing when there are some pollution factors such as perfumes, cleaning products (that contain chemical substances) etc. The measured data is then sent to a ThingBoard server where it can be easily observed the values from the entire month, using diagrams.

The assembly diagram contains:

-ESP8266 module-> the main advantage of this microcontroller is that it can be connected to WiFi.

-Nokia Display 84x48 to display information such as the current time and date, temperature (in degrees Celsius), room's humidity, air quality status (clean, tvoc/eco2 index exceeded, air polluted) and the value of the tvoc & eco2 indices

-DHT11 sensor that measures temperature and humidity from the inside

-CCS811 sensor used for measuring the values for TVOC (Total Volatile Organic Compounds) or ECO2 index.



The data is sent to a ThingsBoard server every 5 minutes. There, graphs can be viewed for each parameter of interest, and the data is displayed over a 7-day period, allowing careful observation of the indices' variations.

Example of Graphs from ThingsBoard




WiFi Connection

If the ESP board detects a network for which it already has saved credentials, then the connection is established without intervention. Otherwise, if no known network is found, a local network called AutoConnectAP will be created. This will open a page based on the local IP address where new credentials (SSID, password) can be entered, which the ESP8266 board will use to connect to WiFi.
